#  React Project Collection

A curated set of beginner-to-intermediate React applications built to explore core concepts, UI design, and component architecture. Each project is self-contained and demonstrates practical use of React (JSX), HTML, and CSS.

---

## 1. Todo App

A task management tool with real-time updates and intuitive controls.

**Frontend:** React (JSX), HTML, CSS  
**Features:**
-  **Task Creation**: Add new tasks with unique IDs and descriptions.
-  **Task Status**: Toggle tasks between completed and pending.
-  **Task Deletion**: Remove individual tasks.
-  **Clear All**: Delete all tasks at once.
-  **Duplicate Prevention**: Prevent adding the same task twice.
-  **Date & Time Display**: Real-time clock shown in the header.

---

## 2. Recipe Finder

A dynamic recipe search app powered by TheMealDB API.

**Frontend:** React (JSX), HTML, CSS  
**Features:**
-  **Recipe Search**: Search recipes by keyword.
-  **Dynamic Results**: Display recipe cards with images, titles, and descriptions.
-  **Detailed View**: Link to full instructions and ingredients.
-  **Error Handling**: Gracefully handles empty queries or failed API requests.
-  **Interactive UI**: Styled search bar, hover effects, and animated buttons.

---

##  3. Rock Paper Scissors Game

An interactive game with score tracking and modular components.

**Frontend:** React (JSX), HTML, CSS  
**Features:**
-  **Player vs Computer**: Random choice generation and outcome logic.
-  **Score Tracking**: Tracks wins, losses, and draws.
-  **Reset Functionality**: Restart the game and clear scores.
-  **Modular Structure**: Components for Game, Controls, Player, Result, and Score.

---
## 4. React Admin Dashboard

A modern, responsive **Admin Dashboard** built with **React, Vite, Tailwind CSS, and Bun**.  
This project showcases professional features like authentication, CRUD operations, charts, tables, and role-based access — designed to match real-world IT company standards.

---

## Features
-  Authentication (Login/Logout, Role-based access)
-  Interactive charts & analytics
-  CRUD operations (Create, Read, Update, Delete)
-  Responsive tables with search & pagination
-  Tailwind CSS styling with dark/light mode
-  Built with Vite for fast development
-  ESLint & PostCSS integration for clean code

---

##  Tech Stack
- Frontend: React, Vite
- Styling: Tailwind CSS, PostCSS
- State Management: React Hooks / Context API (for theme toggle, auth state, global dashboard state)
- Authentication: Custom API (JWT‑based, role‑based access for Admin/User)
- Charts: ApexCharts (react‑apexcharts) for line, pie, and analytics charts
- Package Manager: Bun

---

## Setup

- Navigate into the project:
cd react-admin-dashboard
- Install dependencies:
bun install
- Run the development server:
bun run dev
